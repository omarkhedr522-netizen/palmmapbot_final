--------------------------------------------------------------------------------
Usage Notes
--------------------------------------------------------------------------------

1. The model is trained for aerial/top-down palm tree detection
2. Confidence threshold: 0.25 (default for logging detections)
3. The model performs best on clear, unobstructed palm tree views
4. Lower recall (0.45) indicates some palm trees may be missed
5. For edge deployment, use the NCNN format for better performance
